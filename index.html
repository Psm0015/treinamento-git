Quais os comandos para configurar usuário e email no git?

git config --global user.name "nome-usuario"
git config --global user.email "usuario@dominio.com"

Qual a diferença entre git e github?

Git-é um software instalado na maquina que permite com que o usuário possa salvar versões dos seus projetos

Já o Github é uma plataforma web que absorve os recursos de versões e os compartilha para quem o usuário escolher para que possa ser modificado por várias pessoas simultaneamente

O que faz o comando commit?

O comando commit adiciona os novos progresso do projeto, como se fosse um checkpoint, ele armazenas as atualizações de seu projeto.

O que faz o comando push?

O comando que usamo quando queremos enviar nosso repositório local para um repositório online.
